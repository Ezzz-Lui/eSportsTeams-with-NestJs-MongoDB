# Ejecutar en desarrollo (dev)
1. ### Clonar el repositorio
2. ### Instalar las dependencias (npm o yarn)
```bash
yarn install
```
3. ### Tener NestJs CLI instalado
```bash
npm i -g @nestjs/cli
```
4. ### Base de datos (MongoDB)
```bash
docker-compose up -d
```
